# WormHunt

My first experience at building a Flutter game!

# How to install
1) Install Flutter SDK
2) Install Git for Windows/Linux/Mac
3) Install Microsoft Edge
  All the steps below run in project directory
4) flutter doctor
5) flutter pub get
6) flutter config --enable-web
   flutter doctor -v
7) flutter run -d edge   

# How to play 
A, D to move the tank and only shoot when you have successfully lured the Worm to the TNT
